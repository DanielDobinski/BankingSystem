Short description:
The banking system is using a fixed array size with following data:
ID, name, cash for each owner. ID is unique and created randomly.
You can:
- log in
- withdraw
- deposit
- see current account
- see all acounts
- see accounts' count
- transfer money
- create new account
    accountList is a singletone class which is mainly a list of accounts.
